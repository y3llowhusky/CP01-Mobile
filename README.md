-- Resumo do Projeto --
O aplicativo é um formulário mobile desenvolvido em React Native (com Expo) que permite ao usuário inserir informações pessoais e acadêmicas, como nome, curso, disciplina e uma descrição.

Após o preenchimento, ao clicar no botão “Enviar dados”, os dados digitados são armazenados em estados (useState) e exibidos na tela por meio de um componente separado chamado Review, que apresenta as informações de forma organizada para conferência.

O projeto utiliza conceitos fundamentais como:

Componentização (separação em App e Review)

Gerenciamento de estado com useState

Renderização condicional (mostrar dados apenas após o clique)

Estilização com StyleSheet
